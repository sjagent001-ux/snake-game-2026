# 제품 문서 — snake-game-2026

## 프로젝트 개요

- **이름**: snake-game-2026
- **한 줄 정의**: 설치 없이 URL 하나로 즉시 플레이하는 모던 스네이크 게임 (웹 브라우저 전용)
- **핵심 가치 제안**: 클래식 스네이크의 친숙함 위에 네온 비주얼, 파워업, 부드러운 보간 이동을 더해 캐주얼 플레이어와 포트폴리오 관람자 모두를 만족시키는 경량 웹 게임

## 핵심 기능 (Modern MVP 4가지)

### 1. 파워업 먹이 (Power-up Foods)

맵에 주기적으로 등장하는 특수 아이템. 뱀이 먹으면 제한 시간 동안 한 가지 효과를 부여한다.

- **속도 변화**: 일시적으로 이동 속도를 높이거나 낮춰 난이도를 역동적으로 변화시킨다.
- **점수 2배**: 효과 지속 중 획득하는 모든 점수에 2배 배율을 적용한다.
- **무적**: 벽·장애물·자기 몸과의 충돌 판정을 일시 해제한다.

### 2. 장애물 / 벽 (Obstacles & Walls)

맵 내부에 배치된 벽 타일이 이동 공간을 제한한다. 모든 게임에서 항상 활성화된다. 토글 기능은 본 MVP에 포함하지 않는다.

- **고정 벽**: 매 게임 동일 위치에 존재하는 타일로 플레이어가 레이아웃을 학습할 수 있다.
- **랜덤 벽**: 게임 시작 시 무작위로 배치되어 매 판 다른 공간 안배를 요구한다.

### 3. 부드러운 보간 이동 — Sub-grid Interpolation

격자 좌표를 순간이동하는 방식 대신 tick 사이의 프레임을 선형 보간하여 뱀이 매끄럽게 미끄러지듯 이동한다.

- 고정 tick 스케줄러가 게임 로직을 일정 간격으로 업데이트한다.
- 렌더러는 tick 사이의 경과 시간 비율(alpha)을 받아 위치를 보간하므로 화면 갱신률이 게임 속도에 독립적이다.

### 4. 네온 / 다크 테마 + 파티클 & 트레일 이펙트

게임의 시각적 정체성을 정의하는 비주얼 레이어.

- 다크 배경에 네온 그라데이션으로 칠해진 뱀 몸체
- 뱀 꼬리에 잔광을 남기는 글로우 트레일
- 먹이 섭취 시 발산하는 잔해·섬광 파티클 버스트

## 게임 규칙 요약

| 규칙 | 내용 |
|------|------|
| 이동 | 방향키 또는 WASD로 상·하·좌·우 이동. 반대 방향으로는 즉시 전환 불가. |
| 먹이 | 맵에 항상 일반 먹이 1개 이상 존재. 먹으면 꼬리가 1칸 늘어나고 점수가 증가. |
| 성장 | 뱀 길이는 먹이 섭취마다 1씩 증가. 파워업 먹이 섭취도 성장 적용. |
| 충돌 | 벽(맵 경계·내부 장애물) 또는 자기 몸과 충돌하면 게임 오버. |
| 게임 오버 조건 | 충돌 발생. 무적 파워업 활성 중에는 충돌 판정이 해제되어 오버되지 않음. |

## 상태 머신

```
idle ──(Space)──► running
                    │
         (Space)    │    (Space)
          ◄────── paused ◄──────
                    │
         (충돌 발생)│
                    ▼
                 gameover ──(R 키)──► idle
```

| 상태 | 설명 | 전이 트리거 |
|------|------|-------------|
| idle | 시작 화면 표시, 아무 조작도 없음 | Space → running |
| running | 게임 진행 중. 고정 tick 활성화. | Space → paused, 충돌 → gameover |
| paused | 게임 일시 정지. 렌더는 유지, tick 중단. | Space → running |
| gameover | 게임 오버 화면. 최고점수 저장. | R 키 → idle |

## 점수 시스템

점수 2배 효과가 적용되는 동안 모든 점수 가산값(일반 먹이, 파워업 먹이, 보너스)에 ×2 배율을 곱한다. 예: 파워업 먹이 +15 → +30.

| 이벤트 | 기본 점수 |
|--------|-----------|
| 일반 먹이 섭취 | +10점 |
| 파워업 먹이 섭취 (속도 변화) | +15점 |
| 파워업 먹이 섭취 (점수 2배) | +15점 (이후 획득 점수에 2배 배율 적용) |
| 파워업 먹이 섭취 (무적) | +15점 |
| 점수 2배 효과 중 일반 먹이 섭취 | +20점 (10 × 2) |
| 점수 2배 효과 중 파워업 먹이 섭취 | +30점 (15 × 2) |

- 최고점수는 `localStorage`에 키 `snake2026_highscore`로 저장된다.
- 페이지를 새로고침해도 최고점수는 유지된다.
- 백엔드 없이 해당 기기의 브라우저 스토리지에만 저장된다.

## 비범위 (Non-goals)

본 프로젝트의 정체성과 충돌하여 향후에도 다루지 않는 항목 — 정적 브라우저 Canvas 게임이라는 본질과 양립하지 않으므로 영구 제외한다.

- **백엔드 서버 의존 (계정 / 로그인 / 세션)**: 이 게임은 서버 없이 정적 파일만으로 배포된다. 사용자 계정·세션 관리는 프로젝트 정체성과 구조적으로 충돌한다.
- **결제 또는 광고 기반 수익화**: 포트폴리오·학습 목적의 오픈 소스 프로젝트에 수익화 레이어를 도입하지 않는다.
- **운영체제별 네이티브 앱 빌드**: Electron·Capacitor 등을 통한 데스크톱/모바일 네이티브 패키징은 "URL 하나로 바로 플레이"라는 핵심 가치와 충돌한다.
- **온라인 랭킹 / 글로벌 리더보드**: 점수 저장·조회에 백엔드 서버가 필수적으로 요구되므로 영구 제외한다.
- **멀티플레이어 / 실시간 대전**: 서버 통신(WebSocket 등) 의존이 필수이므로 정적 배포 모델과 양립하지 않아 영구 제외한다.

## 향후 확장 후보

본 MVP에는 포함하지 않으나, 프로젝트의 정적 브라우저 게임 정체성을 깨지 않으면서 별도 SPEC으로 추후 검토 가능한 항목.

1. **사운드 / 음악 (선택적 토글)**: 먹이 섭취·게임 오버 효과음, 배경 BGM 및 음소거 토글
2. **모바일 터치 컨트롤 (스와이프 제스처)**: 스와이프 방향 인식 또는 가상 D-패드 오버레이
3. **다국어 i18n (UI 텍스트 번역)**: 한국어 외 언어(영어, 일본어 등) UI 텍스트 현지화
4. **데일리 챌린지 (날짜 시드 기반 고정 맵)**: 날짜를 시드로 고정된 맵에서 하루 한 번 경쟁
5. **추가 테마 (라이트 / 레트로 등)**: 밝은 계열 테마, 레트로 픽셀 테마 등 ThemeRegistry 확장

## 성공 기준

아래 기준이 모두 충족되어야 MVP가 완성된 것으로 본다.

1. **즉시 실행 가능성**: 배포 URL을 열면 추가 설치 없이 3초 이내에 게임이 로드되어 플레이 가능한 상태가 된다.
2. **60 FPS 렌더링**: Chrome·Edge·Firefox·Safari 최신 2버전 기준으로 requestAnimationFrame 루프가 60 FPS를 안정적으로 유지한다.
3. **전 기능 동작**: 파워업 3종(속도 변화, 점수 2배, 무적), 일반 먹이, 장애물 배치, sub-grid 보간 이동, 파티클 이펙트가 실제 게임 플레이에서 모두 정상 동작한다. 렌더 레이어(파티클, sub-grid 보간)의 시각적 정상성은 수동 시각 검증으로 확인하며 자동화 테스트 범위에 포함하지 않는다.
4. **최고점수 영속성**: 페이지 새로고침 및 탭 재오픈 후에도 localStorage에 저장된 최고점수가 정확히 복원된다.
5. **TDD 커버리지**: Vitest 기준 도메인·엔진 레이어 커밋당 라인 커버리지 80% 이상 유지.
